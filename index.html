Używanie operatora równości

W pythonie (jak i w innnych językach programowania) jest dostępny operator równości (==), który można użyć do sprawdzenia, jak pokazano poniżej

def porownaj_hasla(haslo1, haslo2):
    if haslo1 == haslo2:
        return True
    else:
        return False
